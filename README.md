# codealpha_tasks

Welcome to my *C++ Internship Projects Repository*. This repository contains solutions for the programming tasks assigned during my online internship. The projects demonstrate the fundamentals of C++ programming, object-oriented programming (OOP), file handling, data structures, and algorithm implementation.

---

## 📌 Internship Tasks

### ✅ Task 1: CGPA Calculator

#### Objective
Develop a C++ program to calculate the CGPA of a student.

#### Features
- Accepts the number of courses.
- Takes grade and credit hours for each course.
- Calculates:
  - Total Credits
  - Total Grade Points
  - Semester GPA
  - Overall CGPA
- Displays individual course details.
- User-friendly output.

#### Concepts Used
- Loops
- Arrays/Vectors
- Functions
- Arithmetic Operations
- User Input & Output
